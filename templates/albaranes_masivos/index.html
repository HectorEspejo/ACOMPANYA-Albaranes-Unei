Generación Masiva de Albaranes

Sistema de Generación Automática

Este sistema permite generar albaranes de forma masiva a partir de un archivo CSV con los requisitos de menús para cada cliente.

Formato del archivo CSV:
  • Columna A: Nombre del cliente (en filas separadas)
  • Columna B: Tipo de menú (ej: BASAL INDIVIDUAL ALMUERZO)
  • Columnas C-I: Cantidades para cada día (L, M, X, J, V, S, D)
Funcionamiento:
  1. El sistema leerá el archivo CSV
  2. Creará o buscará los clientes especificados
  3. Generará albaranes para 4 semanas completas
  4. Asignará los menús correspondientes según el tipo de dieta y comida
